#  Analyse Exploratoire — Indicateurs Économiques Africains

##  Description
Analyse exploratoire de données macroéconomiques de 53 pays africains
sur la période 2010-2023, basée sur les données de la Banque Mondiale.

##  Objectifs
- Explorer les indicateurs économiques clés en Afrique
- Visualiser les disparités entre pays
- Identifier les tendances de croissance et d'inflation

##  Visualisations réalisées
- Top 10 pays par PIB par habitant (2023)
- Évolution du PIB du Sénégal (2010-2023)
- Comparaison de l'inflation (5 grands pays)
- Relation entre chômage et PIB par habitant
- Top 10 pays par croissance moyenne du PIB

##  Technologies utilisées
- Python 3.x
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

##  Structure du projet
projet-eda-afrique/

##  Insights clés
- Maurice domine le PIB par habitant en Afrique
- L'Égypte a connu l'inflation la plus instable
- L'Éthiopie affiche la croissance moyenne la plus élevée
- Le chômage en Afrique ne reflète pas toujours la réalité
  économique à cause du secteur informel

##  Source des données
World Bank Open Data via Kaggle
